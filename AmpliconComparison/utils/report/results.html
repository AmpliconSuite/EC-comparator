Results:

Here we compute the cycle distance between `s1` and `s2`.

Distance description:
1. cn_cos_dist = measures the copy-number distance between the s1 and s2 using cosine distance
2. cn_jc_dist = measures the copy-number distance between the s1 and s2 using jaccard index distance
3. cn_hamming_norm_dist = measures the amount of dissimilar genomic bins usage for s1 and s2
4. fragments_dist = measures the amount of fragment usage per genomic bin which is dissimilar between s1 and s2
5. cycles_dist = measures the amount of cycles usage per genomic bin which is dissimilar between s1 and s2
6. breakpoint_dist = measures how many breakpoints-pairs could not be matched between s1 and s2

The radial plot shows all these distances, with values between 0 (low cost, high similarity) and 1 (high cost, low similarity).
The more colorful the more distant are `s1` and `s2`.
